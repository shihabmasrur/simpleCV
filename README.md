# simpleCV
a simple CV maker from information input, the CV design is a standard ATS friendly format. 


## What this is

A basic web app that takes user input and generates a clean, ATS-friendly CV.
If a field is left empty, that section is not included.

## Features

* Simple form-based input
* Auto-generates CV instantly
* Clean, standard layout (ATS-friendly)
* Optional profile photo with crop support
* Download CV as PDF

## How it works

1. User fills out the form (name, education, skills, etc.)
2. App generates a CV preview
3. Empty fields are skipped automatically
4. User downloads the CV as a PDF

## Tech Stack

* HTML
* CSS
* JavaScript

## How to use

1. Open the project
2. Fill in your details
3. Click generate
4. Download your CV
